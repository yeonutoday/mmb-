  KDM_fontcol_data
  KDM_tb_data
  파일의 퍼미션을 666으로 해주세요.

-------------------------------------
스킨내 업데이트사항 : 
option -  이미지업로드버튼 ： 찾아보기 버튼 대신 이미지 버튼을 사용합니다
	   - 상단 메뉴 형식 ： 비툴의 레이아웃에서 메뉴를 상단에 둡니다.
	   - 덧글 쓰는칸 접기 : 덧글 접는 것을 option에 넣었습니다.
	   - 답글 관리자 전용 모드 : 추가된 답글 기능을 관리지만 사용합니다.
	   - 기타 임의로 기능을 삭제 및 제대로 작동되게 수정하였습니다 :]

kd_skin - 기존 사항에서 별반 추가된 게 없지만 변수 정리가 있었습니다 :]


기능 개선 사항 :
-v002에서 판정페이지를 거치지 않으면 비툴 로그가 보이지 않는 문제를 수정하였습니다.
-로그가 100개가 넘어갈 시 테이블이 깨지는 현상을 수정하였습니다.(단, 100개가 넘어갈 시 1페이지에 글쓰기 버튼이 깨져보이는건 고쳐지지 않았습니다.)
-secret 기능에서 member secret기능이 추가되었습니다.
 (메뉴쪽 체크박스 순서대로 more : M / secret : S / member secret : MS)
-태그 업로더 버튼이 생겼습니다.(로그 대신 태그를 적으면 로그로 올라갑니다.)
-이미지 이름이 시간값으로 저장되게 하였습니다.
-reply 기능을 추가하였습니다(수정, 삭제 전부 가능)
-관리자 로그인 시 쿠키가 오래 지속되도록 했습니다.
-관리자의 경우 ad_set의 지정된 암호를 사용하게 했습니다.

-그림 및 dat파일의 비공개를 원하시는 분은 비툴 설치 후 환경변수 설정(맨 처음 실행시 관리자 페이지에서 만들라는거)을 마친 후 
	1. ad_set.php파일을 열어 dat파일과 이미지 파일이 저장될 곳을 각각 다른 이름으로 적는다.(ex  그림저장 : 'pic1111'폴더 / dat저장 : 'log2222'폴더)
	2. 계정에 업로드
	3. 계정내 기존 data폴더를 지정한 dat파일이 저장될 폴더 이름으로 바꾼다.
	4. 그림이 저장될 폴더를 만들거나 혹은 그림이 저장되어있는 폴더 이름을 바꾼다.
